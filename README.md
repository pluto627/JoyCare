# XunDoc 医疗应用

## 项目简介
XunDoc是一个基于SwiftUI开发的iOS医疗应用，旨在为用户提供便捷的医疗服务管理体验。

## 主要功能
- 医院管理
  - 添加和管理医院信息
- 提醒功能
  - 医疗相关提醒事项管理
- 个人档案管理

## 技术栈
- SwiftUI
- iOS 14.0+
- Swift 5.0+

## 项目结构
xundoc2/
├── Views/
│ ├── Profile/
│ │ └── AddHospitalView.swift // 添加医院视图
│ ├── reminder/
│ │ └── ReminderView.swift // 提醒事项视图
│ └── ContentView.swift // 主内容视图

## 安装要求
- Xcode 12.0+
- iOS 14.0+
- macOS Big Sur+
